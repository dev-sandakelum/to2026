Project

NexEra – Full-Stack Academic Learning Platform

A full-stack academic learning platform built for university students to access, organize, and interact with study materials — including Markdown notes, PDFs, and quizzes — all in one place.

Next.js TypeScript Firebase Full-Stack

Problem

University students lacked a centralized, organized platform to access study materials, take quizzes, and track their academic progress across subjects.

Solution

Built NexEra with a Markdown notes system, PDF viewer, real-time quiz engine with auto-save, an admin dashboard for role-based content management, and full mobile responsiveness.

Result

A production-ready academic platform with Firebase-backed server caching, cross-device sync, and Light/Dark/Contrast theme switching — actively serving university students.
